
# Bullet 🔥

- Instructions 🎩

## First 😊:

1. Go to your project Folder 
2. Add bullet.sh 
3. Open your Project in VSCode

## Second 🧐:

1. Open package.json file 
2. Look for   "scripts":
3. Add  "bullet": "./bullet.sh"

```bash
"scripts": {
    "start": "expo start",
    "android": "expo start --android",
    "ios": "expo start --ios",
    "web": "expo start --web",
    "eject": "expo eject",
    "bullet": "./bullet.sh"
  },
```
## Third

1. Open Your Terminal in VSCode 
2. Run
```bash
chmod u+x ./bullet.sh
```
3. Now Run 🤓🔥 
```bash
npm run bullet
```
